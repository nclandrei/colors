# colors

Single-source terminal color palette. Generates theme files for multiple apps from `palette.json`.

## Usage

```bash
uv run generate.py
```

## Targets

| File | App |
|------|-----|
| `themes/ghostty` | Ghostty |
| `themes/iterm2.json` | iTerm2 DynamicProfiles |
| `themes/alacritty.toml` | Alacritty |
| `themes/neovim.lua` | Neovim terminal colors |
